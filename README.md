# Blundersplat-T6ZM
Porte da Blundersplat para todos mapas do zm.

# Como compilar?
Instale a ferramenta [t6modm](https://github.com/Gustavo10Destroyer/t6modm).
Configure as variáveis de ambiente, caso não tenha feito antes, para isso, defina o arquivo `.t6modm.env`.

`t6modm build` - Isso vai gerar o mod para debug, você já pode usar diretamente, mas o indicado é que você
